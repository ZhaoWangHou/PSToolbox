\begin{DoxyItemize}
\item Use Chocolatex for instaaling Unix/\+Mac-\/like programs\+: \href{https://chocolatey.org/}{\texttt{ https\+://chocolatey.\+org/}}
\item Next, install \textquotesingle{}make\textquotesingle{} command by typing\+: {\ttfamily choco install make}
\item You\textquotesingle{}ll also need Min\+GW (Minimalist Gnu for Windows)\+: {\ttfamily choco install mingw}
\item We will also need the eigen package\+: {\ttfamily choco install eigen}
\end{DoxyItemize}

Next you need to clone the repository from Git\+Hub. Once you\textquotesingle{}re done, you need to tailor the Windows makefile {\ttfamily makefile\+\_\+win.\+mk}\+:


\begin{DoxyItemize}
\item Specify the path to Eigen, e.\+g. {\ttfamily INC = -\/IC\+:/\+Program\+Data/chocolatey/lib/eigen/include}
