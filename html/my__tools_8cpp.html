A 2D Nelder-Mead optimum searcher (for illustratory purposes) DO NOT USE!

A 2D Nelder-Mead optimum searcher. Accepts an any dimensional (but will be treated as 2) parametric function, along with a starting point, radius and of course the parameters. There is no real output, only the convergence is displayed on the console, with the solution always running for a 100 iterations.

Parameters

[in]xThe starting (centre) point
[in]parsThe parameters necessary for evaluating the function

An implementation of the multidimensional Newton-Raphson method.

This function finds the root of a function from a given starting point using the multi-dimensional Newton-Raphson method. The implementation also allows for the use of a VectorXd of parameters. The maximum available error and number of iterations also need to be declared.

Returns
Whether the search forthe solution was succesful
