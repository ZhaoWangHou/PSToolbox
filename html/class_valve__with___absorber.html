void Valve_with_Absorber::SetAeffCoeffs ( const double  _a1,
const double  _a2,
const double  _a3,
const double  _AeffMax 
)

Set two elements of a second-order function (and a maximum value) for the relative multiplicator of the effective area. The approximating function will be $A_{eff}=min(A_{eff,max},1+a_1 y+a_2 y^2+a_3 y^3)$, where x is the relative lift: $y=x/(D/4)$

Parameters
_a1[in] first order parameter
_a2[in] second order parameter
_Aeffmax[in] maximum area relative to valve cross section (Abore)
